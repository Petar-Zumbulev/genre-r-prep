# Gen Re R Prep

Preparation repo for a potential working student analyst/coder role at
Gen Re.

## Goal

My goal is to become interview-ready by the end of March for an R-heavy
insurance analytics role focused on practical work such as:

-   R programming
-   tidyverse / data wrangling
-   reporting and visualization
-   ggplot2
-   Shiny apps
-   Excel integration
-   Git workflow
-   SQL / Postgres basics
-   PDF / OCR / API workflows
-   practical insurance analytics

## Role focus

Based on information from a current working student in the role, this
position appears to be more focused on:

-   practical R work
-   data cleaning
-   reporting
-   automation
-   tool-building
-   dashboards
-   Excel-connected workflows

and less focused on deep theoretical actuarial math.

## Main preparation targets

By the end of March, I want to:

1.  write clean and confident R code
2.  become comfortable with common analyst workflows in R
3.  build portfolio projects relevant to insurance analytics
4.  prepare for likely interview tasks such as:
    -   data cleaning
    -   reporting
    -   dashboarding
    -   basic modeling
    -   explaining business logic clearly

## Planned portfolio projects

### 1. Insurance Reporting Dashboard in R/Shiny

A practical dashboard project that will include: - claims / premium
style data - data cleaning - quarterly summaries - severity and trend
analysis - filters and plots - exportable reporting outputs

### 2. PDF / OCR / API to Structured Reporting Workflow

A smaller workflow project that will include: - reading unstructured or
semi-structured input - extracting useful information - converting it
into a clean table or report - exporting structured output
